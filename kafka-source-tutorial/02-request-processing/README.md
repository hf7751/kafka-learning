# 02. иҜ·жұӮеӨ„зҗҶжңәеҲ¶

## жң¬з« еҜјиҜ»

иҜ·жұӮеӨ„зҗҶжҳҜ Kafka зҡ„ж ёеҝғеҠҹиғҪпјҢиҙҹиҙЈжҺҘж”¶гҖҒеӨ„зҗҶе’Ңе“Қеә”е®ўжҲ·з«ҜиҜ·жұӮгҖӮжң¬з« иҠӮж·ұе…ҘеҲҶжһҗ Kafka зҡ„зҪ‘з»ңеұӮжһ¶жһ„гҖҒиҜ·жұӮеӨ„зҗҶжөҒзЁӢе’ҢжҖ§иғҪдјҳеҢ–жңәеҲ¶гҖӮ

## ж–ҮжЎЈзӣ®еҪ•

| ж–ҮжЎЈ | еҶ…е®№ | йў„и®Ўйҳ…иҜ»ж—¶й—ҙ |
|------|------|-------------|
| [01-socketserver.md](./01-socketserver.md) | SocketServer зҪ‘з»ңеұӮж·ұеәҰи§Јжһҗ | 45 еҲҶй’ҹ |
| [02-request-handler.md](./02-request-handler.md) | KafkaApis иҜ·жұӮеӨ„зҗҶжһ¶жһ„ | 40 еҲҶй’ҹ |

## еӯҰд№ и·Ҝеҫ„е»әи®®

### еҝ«йҖҹдәҶи§Ј (30еҲҶй’ҹ)
```
1. SocketServer зҪ‘з»ңеұӮи§Јжһҗ
2. KafkaApis иҜ·жұӮеӨ„зҗҶ
```

### ж·ұе…ҘзҗҶи§Ј (1.5е°Ҹж—¶)
```
1. SocketServer зҪ‘з»ңеұӮи§Јжһҗ - зҗҶи§Ј Reactor жЁЎејҸ
2. KafkaApis иҜ·жұӮеӨ„зҗҶ - зҗҶи§ЈиҜ·жұӮи·Ҝз”ұжңәеҲ¶
```

## иҜ·жұӮеӨ„зҗҶжөҒзЁӢйҖҹжҹҘ

```
е®ўжҲ·з«ҜиҜ·жұӮеӨ„зҗҶжөҒзЁӢ

1. е»әз«ӢиҝһжҺҘ
   в””в”Җв”Җ Acceptor жҺҘеҸ—ж–°иҝһжҺҘ
   в””в”Җв”Җ еҲҶй…Қз»ҷ Processor

2. иҜ»еҸ–иҜ·жұӮ
   в””в”Җв”Җ Processor д»Һ Socket иҜ»еҸ–
   в””в”Җв”Җ и§ЈжһҗиҜ·жұӮеӨҙ
   в””в”Җв”Җ е®Ңж•ҙиҜ»еҸ–иҜ·жұӮдҪ“

3. иҜ·жұӮе…Ҙйҳҹ
   в””в”Җв”Җ е°ҒиЈ…жҲҗ RequestChannel.Request
   в””в”Җв”Җ ж”ҫе…Ҙ requestQueue

4. иҜ·жұӮеӨ„зҗҶ
   в””в”Җв”Җ Handler зәҝзЁӢд»ҺйҳҹеҲ—еҸ–еҮә
   в””в”Җв”Җ и°ғз”Ё KafkaApis еӨ„зҗҶ
   в””в”Җв”Җ жү§иЎҢдёҡеҠЎйҖ»иҫ‘

5. еҸ‘йҖҒе“Қеә”
   в””в”Җв”Җ е°ҒиЈ…жҲҗ RequestChannel.Response
   в””в”Җв”Җ ж”ҫе…Ҙ responseQueue
   в””в”Җв”Җ Processor ејӮжӯҘеҸ‘йҖҒ

6. е…ій—ӯиҝһжҺҘ
   в””в”Җв”Җ иҝһжҺҘи¶…ж—¶жҲ–й”ҷиҜҜ
   в””в”Җв”Җ дјҳйӣ…е…ій—ӯ
```
